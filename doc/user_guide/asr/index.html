INTRODUCTION

Welcome to Cerence® ASR, Cerence’s state of the art API for automatic speech recognition (ASR) on embedded platforms. This manual helps you develop applications with the Cerence SDK.

Before starting development, make sure you read the Cerence SDK overview for a general understanding of the Cerence ASR engine and about Cerence SDK. Familiarity with speech recognition is not a prerequisite but is recommended.

This manual is part of the larger Cerence SDK which gives you an integrated offering of Cerence embedded speech and language technology.

Structure of this manual

This manual describes the Cerence ASR features and how to use them. The manual has three parts:

  • Overview of the Cerence ASR product and its most important features.
  • Introduction to some of the essential concepts of building a speech recognition application and how Cerence ASR can help you with that.
  • Introduction to the set of sample codes available in the Cerence SDK.

Contacting Cerence

If you have a support contract, all comments and questions regarding the Cerence ASR SDK are processed through the online call tracker of Cerence’s technical support group. Please see http://distro.cerence.com and we will promptly address your problem. If you wish to get a support contract from Cerence, please contact your account manager.

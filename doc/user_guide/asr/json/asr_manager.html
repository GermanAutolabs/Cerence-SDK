ASR Manager Configuration Options

OVERVIEW

An Asr Manager configures VoconHigh 5 internals. That is, it can configure a multi processing system using Actors as the execution units and events as the messaging mechanism.

In this document, we provide a brief overview of asr manager and their options.

COMMON OPTIONS

Every asr manager type is built up following this schema:

{
  "version" : "VoConHigh 5.0",
  "asr_manager" : [{
   "name": "asrRecognizer",
   "liquid_config_file_name": "liquid_config.json"
  }]
}
name
Uniquely identifies the asr manager.
liquid_config_file_name
The absolute path to the liquid config file

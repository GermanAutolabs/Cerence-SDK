IMp3AudioDecoderConfig - The ‘mp3 decoder’ configuration.

The ‘mp3 decoder’ configuration.

Currently there are no configuration parameters defined.

This built-in mp3-decoder implementation can be bound to an audio decoder module by setting the param “decoder_type” to the value “MP3”.

It transforms audio data with the encoding “ENCODING_MP3” into audio data with the encoding “ENCODING_PCM_16BIT”.

Scope Of Document

The main scope of this SDK documentation is to serve as user guide for engineers who use Cerence SDK to develop and deliver project-specific applications.

Overview

This document describes the speech output (Prompter) component of the Cerence SDK. The SDK Prompter API provides a high-level API abstraction on top of the Cerence Text-To-Speech (TTS) engine providing access to both Cerence TTS Embedded and Cerence Cloud TTS.

In the following sections we give an overview of the high-level Prompter interface that the SDK provides on top of the embedded and cloud TTS engines. This user guide focuses on describing the basic usage and concepts of the Prompter API.

For the full description of prompter interfaces see the API reference documenation of the SDK Prompter component which is included in the Cerence SDK package at doc/api_reference/prompter/. There is a separate folder for each supported programming language in which the reference documentation can be accessed via the mainpage index.html.

The C API documentation is available at doc/api_reference/prompter/c/index.html.

In addition, the SDK package contains a Prompter sample program and sample configurations that illustrate how to use the API. Together with the overview in this document the sample program will be a good starting point for your own implementations.

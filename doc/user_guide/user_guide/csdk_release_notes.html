Known Issues

Ticket - 190638:

There is a known issue with the multiple recognizer use case. The crash is possible when at least one SEM3 Application is added to more than one Recognizer. This issue will be fixed in the next release.

Ticket - 190585:

There is a known issue with the runtime validation of JSON configuration files in the ASR component. When the validate parameter of nuance_common_IConfiguration_create<*>WithValidation API calls is set to true, then valid JSON configuration files for ASR can no longer be parsed and API calls return with an error.

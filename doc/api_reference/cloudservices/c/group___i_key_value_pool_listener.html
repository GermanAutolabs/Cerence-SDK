typedef struct _nuance_common_IKeyValuePoolListener nuance_common_IKeyValuePoolListener

Provides callback methods to receive change notifications from the keyValuePool object.

This listener is intended for DragonDriveFramework use only.

typedef void(* FUNC_nuance_common_IKeyValuePoolListener_onChange) (void *instance, nuance_common_IKeyValuePool *keyValuePool, const LH_CHAR *key, const LH_CHAR *paramValue)

This function returns the changed value for the given key.

Parameters
[in,out]instanceThe instance pointer
[in]keyValuePoolInstance of the KeyValuePool object.
[in]keyKey for which notification was requested, that was just changed.
[in]paramValueNew value stored for key.
typedef void(* FUNC_nuance_common_IKeyValuePoolListener_onRemove) (void *instance, nuance_common_IKeyValuePool *keyValuePool, const LH_CHAR *key)

This function notifies that the registered key was removed.

Parameters
[in,out]instanceThe instance pointer
[in]keyValuePoolInstance of the KeyValuePool object.
[in]keyKey for which notification was requested, that was just removed.
typedef void(* FUNC_nuance_common_IKeyValuePoolListener_onError) (void *instance, nuance_common_KeyValuePoolError error, const LH_CHAR *message)

This function is called each time an error occurs in the KeyValuePool object. This will only happen if the KeyValuePool was created with a configuration. Every failure of validating requests against the possible keys and value ranges will result in a call to this method.

Parameters
[in,out]instanceThe instance pointer
[in]errorThe type of error that occurred
[in]messageString describing the error

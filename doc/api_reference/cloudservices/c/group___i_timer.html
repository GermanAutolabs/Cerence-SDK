Function Documentation

nuance_common_ResultCode nuance_common_ITimer_start ( nuance_common_ITimer *  instance,
LH_S32  timeout,
nuance_common_ITimerListener *  listener 
)

Start the timer.

This method starts the timer with the given timeout (in milliseconds). After the timeout is elapsed, the onTimeout event of the ITimerListener will be called. If a timer was already started on this ITimer instance then ERROR will be returned.

Parameters
[in,out]instanceThe instance pointer
[in]timeoutTimeout value of the timer.
[in]listenera ITimerListener object
nuance_common_ResultCode nuance_common_ITimer_stop ( nuance_common_ITimer *  instance)

Stop the timer.

This method stops the timer. Applications can stop the timer with this method if waiting for the onTimeout event is no longer needed, because another event occured before the timer elapsed. If no timer was started on this ITimer instance then ERROR will be returned.

[in,out]instanceThe instance pointer

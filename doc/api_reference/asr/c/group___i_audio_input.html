Typedef Documentation

typedef struct _nuance_audio_IAudioInput nuance_audio_IAudioInput

Interface of the 'AudioInput' audio module.

The 'AudioInput' audio module captures audio from an IAudioInputAdapter interface implementation. The AudioInput will request an IAudioInputAdapter of the configured type during creation at the audio manager. Optionally such an adapter implementation can also be passed at creation time to the AudioInput.

A valid adapter implementation needs to be provided, there is no default implementation.

nuance_common_ResultCode nuance_audio_IAudioInput_registerFactory ( nuance_audio_IAudioManager *  audioManager)

enables the implicit creation feature for this module at the handed audio manager instance.

This method enables the implicit creation feature for this module at the handed audio manager instance. It takes care of all necessary internal steps to do that, including the creation of the needed factory as well as the registration of that factory object at the audio manager.

The ownership of the factory object is handed to the audio manager by calling this method. It will be destroyed once the audio manager is destroyed.

This feature can be activated only once and stays active as long as the handed audio manager instance is available. Trying to activate that feature a second time will result in an error.

nuance_common_ResultCode nuance_audio_IAudioInput_getPadNames ( const nuance_audio_IAudioInput *  instance,
nuance_common_StringList **  sinkPads,
nuance_common_StringList **  sourcePads 
)

Returns the sink and source pad name(s) of this SDK audio module.

The pad names are given in the configuration of this audio module. In case this audio module does only have one single sink respectively source pad the pad name is equal to the instance name of this module. The pad names have to be unambiguous as they are used as identifier of a specific pad. They are used in conjunction with the instance name to establish/abolish pad connections. The pad name order in the list corresponds the descending order of the sink resp. source pads of this audio module.

This method is thread safe.

Parameters
[in,out]instanceThe instance pointer
[out]sinkPadsSink pad name list of this audio module.
[out]sourcePadsSource pad name list of this audio module.

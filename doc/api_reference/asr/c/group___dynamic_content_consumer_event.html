Enumeration Type Documentation

enum nuance_asr5_DynamicContentConsumerEvent

This enumeration type describes the events which can occur on a dynamic content consumer (dcc).

Enumerator
NUANCE_ASR5_DCC_DATA_PREPARATION_STARTED 

Raised when the dcc data preparation is started.

NUANCE_ASR5_DCC_DATA_PREPARATION_FINISHED 

Raised when the dcc has finished creating its dynamic data. The message contains the current total number of entries, and number of added, removed, failed entries

NUANCE_ASR5_DCC_CREATION_STARTED 

Raised when the dcc creation is started.

NUANCE_ASR5_DCC_CREATION_FINISHED 

Raised when the dcc creation has finished.

NUANCE_ASR5_DCC_CREATION_FAILED 

Raised when the dcc creation failed. Look into the log file for the details of the failure.

NUANCE_ASR5_DCC_WITH_N_ENTRIES 

Raised with the DCC entries count after the dcc creation succeeded.

NUANCE_ASR5_DCC_DATA_ID 

Raised with the DCC data id after the dcc creation succeeded.

NUANCE_ASR5_DCC_N_ENTRIES_PROCESSED 

Raised every time a configured amount (progress_increment) of entries has been processed.

enum nuance_asr5_RecogEvent

This enumeration type describes the events which can occur during a recognizer run.

Enumerator
NUANCE_ASR5_RECOGNIZER_AUDIOSTREAM_RECEIVED 

recognizer received the audio stream

NUANCE_ASR5_RECOGNIZER_SPEECH_DETECTED 

recognizer detected speech

NUANCE_ASR5_RECOGNIZER_SILENCE_DETECTED 

recognizer detected silence

NUANCE_ASR5_RECOGNIZER_AUDIO_STOPPED 

recognizer detected audio stopped

NUANCE_ASR5_RECOGNIZER_AUDIO_FINISHED 

recognizer detected audio finished

NUANCE_ASR5_RECOGNIZER_ABNORMCOND_BADSNR 

The signal to noise ratio is too low.

NUANCE_ASR5_RECOGNIZER_ABNORMCOND_OVERLOAD 

The speech level is too loud.

NUANCE_ASR5_RECOGNIZER_ABNORMCOND_TOOQUIET 

The speech level is too weak.

NUANCE_ASR5_RECOGNIZER_ABNORMCOND_NOSIGNAL 

No or very low input signal.

NUANCE_ASR5_RECOGNIZER_EXTERNAL_PROVIDER_STARTED 

external provider started to work.

NUANCE_ASR5_RECOGNIZER_EXTERNAL_PROVIDER_FINISHED 

external provider finished processing audio.

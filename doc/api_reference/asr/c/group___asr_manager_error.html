Enumeration Type Documentation

enum nuance_asr5_AsrManagerError

This enumeration type describes the errors which can occur in the AsrManager (incl. VoCon5 errors)

Enumerator
NUANCE_ASR5_ASRMANAGER_VOCON_ALLOCATION_FAILURE 

Indicates that an error occurred during memory allocation.

NUANCE_ASR5_ASRMANAGER_VOCON_INVALID_STATE 

Indicates that a vocon object is not in a porper state.

NUANCE_ASR5_ASRMANAGER_VOCON_INVALID_ARGUMENT 

Indicates that an invalid argument was passed to a function.

NUANCE_ASR5_ASRMANAGER_VOCON_INVALID_DATA 

Inidcates that a stream contains invalid data.

NUANCE_ASR5_ASRMANAGER_VOCON_INVALID_VERSION 

Inidcates that the givin data has the wrong version.

NUANCE_ASR5_ASRMANAGER_VOCON_CALLBACK_ERROR

NUANCE_ASR5_ASRMANAGER_VOCON_LIMIT 

Indicates that an internal varialbe has exceeded its maximun value.

NUANCE_ASR5_ASRMANAGER_VOCON_NOTSUPPORTED 

Indicates that a requested feature is not supported.

NUANCE_ASR5_ASRMANAGER_VOCON_ERROR

NUANCE_ASR5_ASRMANAGER_VOCON_STREAMCONFIGBUILDER

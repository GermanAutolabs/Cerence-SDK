Cerence SDK Asr5 component

This document describes the C application programming interface of the Cerence SDK Asr5 component.

The C functions and types of the Asr5 component are prefixed:

  • nuance_asr5_...    Provides automatic speech recognition (ASR) functionalities.

Result codes

C functions of the Cerence SDK return a nuance_common_ResultCode that indicates if an action passed or failed. Further return values need to be passed as return-parameter. If a function returns an object its ownership is transferred to the application.

Many interface functions are implemented asynchronous. Receiving an OK result in that case only means that the call was accepted and will be executed. The result of the functionality itself will be signaled later on with a listener callback.

String and List types

Strings and list types are also returned as objects, i.e. they must be destroyed by the user if they are no longer needed.

The data can be accessed with methods like nuance_common_String_c_str(), nuance_common_StringList_get(), the returned C string must not be freed, it remains valid as long as the corresponding string / list object exists.

Listener callback interfaces

Some SDK components provide listener callback interfaces that can be used optionally in the application. This is useful to get informed about events, warnings, errors, results or such the like. The functions of a listener callback interface are grouped in abstract classes. To get an instance of such a class, a data struct with function pointers must be passed.

Example:

/* implement IRecognizerListener callback interface: */
static void onEvent(void *instance, nuance_asr5_RecogEvent event, LH_S32 timeMarker, const LH_CHAR *message)
{
/* user-defined implementation ... */
}
static void onResult(void *instance, const LH_CHAR *result, nuance_asr5_ResultType resultType, LH_BOOL isFinal)
{
/* user-defined implementation ... */
}
/* provide implementations of all the other IRecognizerListener callback functions here ... */
vtable_nuance_asr5_IRecognizerListener recognizerListenerCb = { onEvent, onResult, ... };
/* create IRecognizerListener object, based on user-defined callback functions */
nuance_asr5_IRecognizerListener *recognizerListener = 0;
recognizerListener = nuance_asr5_IRecognizerListener_create(&recognizerListenerCb, 0 /* user data */);
...

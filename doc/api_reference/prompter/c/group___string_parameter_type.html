enum nuance_prompter_StringParameterType

This enumeration describes String parameter types of the prompter.

Enumerator
NUANCE_PROMPTER_TEXT_NORMALIZATION_TYPE 

Describes the text normalization type (TN type). It overrules the value of the "text_normalization_type" key defined in the currently activated usecase.

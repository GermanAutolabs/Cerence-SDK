typedef struct _nuance_cloudservices_ICloudCommanderListener nuance_cloudservices_ICloudCommanderListener

Callback interface to get notified about the completion of async interface functions.

This class declares an interface which can be implemented by the customer to get notified when different asyncronous interface functions of ICloudCommander are finished.

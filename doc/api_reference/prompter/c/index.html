Cerence SDK Prompter component

This document describes the C application programming interface of the Cerence SDK Prompter component.

The C functions and types of the Prompter component are prefixed:

  • nuance_prompter_...    Provides Text-To-Speech conversion functionalities.

Result codes

C functions of the Cerence SDK return a nuance_common_ResultCode that indicates if an action passed or failed. Further return values need to be passed as return-parameter. If a function returns an object its ownership is transferred to the application.

Many interface functions are implemented asynchronous. Receiving an OK result in that case only means that the call was accepted and will be executed. The result of the functionality itself will be signaled later on with a listener callback.

String and List types

Strings and list types are also returned as objects, i.e. they must be destroyed by the user if they are no longer needed.

The data can be accessed with methods like nuance_common_String_c_str(), nuance_common_StringList_get(), the returned C string must not be freed, it remains valid as long as the corresponding string / list object exists.

Listener callback interfaces

Some SDK components provide listener callback interfaces that can be used optionally in the application. This is useful to get informed about events, warnings, errors, results or such the like. The functions of a listener callback interface are grouped in abstract classes. To get an instance of such a class, a data struct with function pointers must be passed.

Example:

/* implement IPrompterListener callback interface: */
static void onError(void *instance, nuance_prompter_PrompterError error, const char *msg)
{
printf("Error occurred: %s - %s\n", nuance_prompter_PrompterError_toString(error), msg);
}
static void onWarning(void *instance, nuance_prompter_PrompterError warning, const char *msg)
{
printf("Warning occurred: %s - %s\n", nuance_prompter_PrompterError_toString(warning), msg);
}
static void onTextSegment(void *instance, const char *text)
{
}
static void onAnalyzeTextFinished(void *instance)
{
}
static void onCreated(void *instance, nuance_common_ResultCode rc)
{
printf("IPrompter instance creation finished with result: %s\n", nuance_common_ResultCode_toString(rc));
}
vtable_nuance_prompter_IPrompterListener prompterListenerCb = { onError, onWarning, onTextSegment, onAnalyzeTextFinished, onCreated };
/* create IPrompterListener object, based on user-defined callback functions */
nuance_prompter_IPrompterListener *prompterListener = 0;
prompterListener = nuance_prompter_IPrompterListener_create(&prompterListenerCb, 0 /* user data */);

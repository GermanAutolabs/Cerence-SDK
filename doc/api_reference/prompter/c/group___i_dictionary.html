typedef struct _nuance_common_IDictionary nuance_common_IDictionary

IDictionary storage class.

This class allows to store key-value pairs of different data type.

const LH_CHAR *  key,
const nuance_common_IDictionary *  data 
)

Add a key/data pair of type IDictionary.

This mehtod adds an IDictionary instance to this Dictionary. The hosting Dictionary internally increases the reference count of the "Guest-Dictionary", which can be deleted or modified directly after the put call. Put of an already existing key, will replace the old dictionary through the new one.

Parameters
[in,out]instanceThe instance pointer
[in]keyKey under which the data should be inserted.
[in]dataIDictionary instance to store.

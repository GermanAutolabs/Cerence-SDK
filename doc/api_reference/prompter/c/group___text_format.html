Detailed Description

Enumeration Type Documentation

enum nuance_prompter_TextFormat

This enumeration describes the format of the input text that has to be read out with the prompter. Depending on the format, the prompter component may preprocess given input text and convert it to the form most appropriate for the audio synthesis.

Enumerator
NUANCE_PROMPTER_FORMAT_PLAIN 

The plain text input format.

Defines plain input text that requires no special preprocessing by the prompter.

NUANCE_PROMPTER_FORMAT_EMAIL 

The email text input format.

This format is intended for CDFW applications only. Defines input text formatted as an email content. This input text must be in the required email format.

NUANCE_PROMPTER_FORMAT_SSML 

The SSML (Speech Synthesis Markup Language) text input format.

Defines input text formatted as SSML content.

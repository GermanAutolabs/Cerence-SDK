typedef struct _nuance_common_ILogConsumer nuance_common_ILogConsumer

Abstract interface implemented by the customer to receive log messages.

typedef nuance_common_ResultCode(* FUNC_nuance_common_ILogConsumer_configure) (void *instance, const LH_CHAR *params)

Configures this consumer.

This method configures a consumer with a specific configuration string in JSON format. The possible content of the JSON string depends on the configuration parameters this consumer supports. The JSON configuration string passed to this method can be set with the params parameter of the ILoggingConsumerConfig in the logger JSON configuration file. In case the params parameter is not set or empty this method is called anyway. This method is called before any other method of this interface.

Parameters
[in,out]instanceThe instance pointer
[in]paramsconsumer specific configuration parameter json string.

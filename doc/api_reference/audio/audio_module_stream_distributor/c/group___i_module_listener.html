typedef struct _nuance_common_IModuleListener nuance_common_IModuleListener

Callback interface to get notified about changes of the module state.

This class declares an interface which can be implemented by the customer to get notified about different state changes of an IModule object.

typedef void(* FUNC_nuance_common_IModuleListener_onCreated) (void *instance, nuance_common_ResultCode resultCode)

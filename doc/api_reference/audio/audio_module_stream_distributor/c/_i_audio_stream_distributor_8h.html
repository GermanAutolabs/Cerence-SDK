Detailed Description

Interface of the 'AudioStreamDistributor' audio module.

The AudioStreamDistributor (abbr.'ASD') module is capable to map the audio channels of one input stream to one or more output streams. Means in detail, the ASD is able to reorder, to pick and to fully duplicate the audio channels of one input stream to one or more output streams.

Detailed Description

Listener interface of the 'AudioManager' component.

Vtable for providing overridden class functions.

The AudioManager component is one of the central elements of the audio backbone and will act as a proxy to collect audio related events of the audio backbone, offering those via its listener interface.

FUNC_nuance_audio_IAudioManagerListener_onSseSignalStatistic onSseSignalStatistic

This event listener method is called via the audio manager component in the interval the specific instance was configured for.

This method is triggered by the speech signal enhancement module if the parameter "signal_statistics_interval" is configured.

Parameters
[in,out]instanceThe instance pointer
[in]instNamewho is emitting this event
[in]eventDatadata container holding several properties for each channel of a stream

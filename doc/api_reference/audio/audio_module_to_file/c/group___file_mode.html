enum nuance_audio_FileMode

Define the file open mode.

This enumeration represents the mode in which the file is opened.

Enumerator
NUANCE_AUDIO_FM_OVERWRITE 

will create a new file if not present or overwrite the file if it already exists

NUANCE_AUDIO_FM_APPEND 

will append the new data to an already existing file

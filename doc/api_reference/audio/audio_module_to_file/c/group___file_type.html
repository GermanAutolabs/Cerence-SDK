enum nuance_audio_FileType

Define the content type of the audio file.

This enumeration lists supported content types of the audio file.

Enumerator
NUANCE_AUDIO_FT_RAW 

Raw audio data (either signed 16Bit PCM or encoded)

NUANCE_AUDIO_FT_WAV 

Signed 16Bit pcm raw data with WAV header

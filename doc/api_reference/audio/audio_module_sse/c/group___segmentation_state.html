enum nuance_audio_SegmentationState

This enumeration type describes the status which can occur during a audio segmentation run.

Used at the "onSegmentation" method of the IAudioManagerListener. This method is triggered by the segmentation module if added to the audio path. This module is only available at the DDFW.

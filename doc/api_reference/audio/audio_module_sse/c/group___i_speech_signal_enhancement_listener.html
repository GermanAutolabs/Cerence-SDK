typedef struct _nuance_audio_ISpeechSignalEnhancementListener nuance_audio_ISpeechSignalEnhancementListener

Callback interface to get notified about the completion of asynchronous interface functions.

This class declares an interface which can be implemented by the customer to get notified when different asynchronous interface functions of ISpeechSignalEnhancement are finished.

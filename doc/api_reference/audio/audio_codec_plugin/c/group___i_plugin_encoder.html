nuance_common_ResultCode nuance_audio_IPluginEncoder_registerFactory ( nuance_audio_IAudioManager *  audioManager)

Registers a plugin encoder.

This method loads the shared object of a specific plugin encoder from the file system and registers it at the audio manager. With that an Encoder module instance is able to request and use an instance of the audio encoder provided by this specific plugin.

A plugin encoder has to implement the API defined in the header file IEncoderPlugin.h.

Parameters
[in]audioManagerAudio manager instance.

nuance_common_ResultCode nuance_common_IDiagnosis_storeKeyValue ( nuance_common_IKeyValuePool *  keyValuePool,
const LH_CHAR *  key,
const LH_CHAR *  paramValue 
)

This API call allows applications (GDML dialogs and/or external diagnosis programs) to store general information for diagnosis purposes (For setting config params use API storeConfigKeyValue() instead). Writes the value for the given key to the DDFW KeyValuePool "dd_diag_key_value_pool". Possible keys need to be defined in application-specific configurations. If the key or the value cannot be validated against the configuration an error will be signalled if the validation fails (as result and by a call to the onError method if a listener was provided on creation).

Parameters
[in]keyValuePoolInstance of the KeyValuePool object (dd_diag_key_value_pool).
[in]keyThe name of the general key for which the value should be set.
[in]paramValueThe new value to be set.

nuance_common_ResultCode nuance_common_IDiagnosis_storeConfigKeyValue ( nuance_common_IKeyValuePool *  keyValuePool,
const LH_CHAR *  key,
const LH_CHAR *  paramValue 
)

This API call allows diagnosis programs to change the value of dynamic configuration parameters (defined as vh_callback in JSON configs). Writes the value for the given key to the DDFW KeyValuePool "dd_common_key_value_pool". Possible keys are vh_callback parameters from the JSON configuration. Values will be read from "dd_common_key_value_pool" next time a DDFW application accesses the vh_callback parameter.

Parameters
[in]keyValuePoolInstance of the KeyValuePool object (dd_common_key_value_pool).
[in]keyThe name of the config key for which the value should be set.
[in]paramValueThe new value to be set.

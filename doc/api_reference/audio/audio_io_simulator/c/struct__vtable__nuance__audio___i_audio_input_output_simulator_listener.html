Detailed Description

Callback interface to get notified about the completion of asynchronous interface functions.

Vtable for providing overridden class functions.

This class declares an interface which can be implemented by the customer to get notified when different asynchronous interface functions of IAudioFrom file are finished.

Field Documentation

FUNC_nuance_audio_IAudioInputOutputSimulatorListener_onFileSet onFileSet

This function is called when setting of a file is finished. It is the asynchronous response to the IAudioInputOutputSimulator::setFile call. When this is called the file was opened and the 1rst 30 chunks were buffered into memory.

Detailed Description

Interface of the 'DecoderFactory' component.

Vtable for providing overridden class functions.

The AudioDecoderFactory is used by the audio decoder module to implement the feature of using different implementations of the IAudioDecoder interface.

The factory needs to be known a the audio manager (see "registerAudioDecoderFactory").

FUNC_nuance_audio_IDecoderFactory_getDecoderType getDecoderType

Returns the type identifier of the audio decoders created, for instance "OPUS"

This is the value which needs to be set at the "decoder_type" property of the decoder module to trigger the creation of such a decoder.

FUNC_nuance_audio_IDecoderFactory_releaseFactory releaseFactory

Release this instance of the factory. No further calls to the interface of this factory will be done after that call. The audio manager has removed that factory handle from its internal registry.

The implementation of that factory can be destroyed.

Since the audio manager calls this releaseFactory method in the context of its destructor, the deletion of the instance is probably the best solution in the majority of the use cases.

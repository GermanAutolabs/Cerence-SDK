Detailed Description

Listener interface of the audio input adapter.

This interface is used from an audio input adapter interface implementation to:

1) announce captured audio packages from the system's audio device to the audio framework of the SDK 2) retrieve the flow status of the SDK's audio framework

The listener interface is implemented by the AudioInput audio module.

Detailed Description

Interface of the 'AudioOutput' audio module.

The 'AudioOutput' audio module streams audio data to an IAudioOutputAdapter interface implementation. The AudioInput will request an IAudioInputAdapter of the configured type during creation at the audio manager. Optionally such an adapter implementation can also be passed at creation time to the AudioOutput.

A valid adapter implementation needs to be provided, there is no default implementation.
